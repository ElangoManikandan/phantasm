Query Quest
DESCRPTION

The event consists of two rounds: SQL Quiz - Test your SQL knowledge with multiple-choice questions. Top scorers move to the next round. Query Execution - Solve real-world SQL problems by executing queries that must pass predefined test cases.

RULES
  1. Individual Participations only
  2. Both rounds are Online mode
  3. First round contains 30 mcqs and it is elimination round
  4. Second round consists of three levels of complexity (easy,moderate,hard)
  5. Avoid malpractices( switch tabs, copy &paste etc.,)
